This was my first project using Parallax. I always thought it was really exciting and finally did it myself!

I'm really in to Astronomy, so I wanted to combine 2 hobbies in one, coding and Astronomy.
The information about the planets is imported from a Planet API. It will update automatically.
I used GSAP for the scroll-ish effect at the bottom. I used this technology in the past but now used it in a public web appliction.
I hope that I can update the website with more information from the API.
