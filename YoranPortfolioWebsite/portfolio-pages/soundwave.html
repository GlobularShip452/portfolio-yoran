Soundwave - Music site

Yoran Essers

Fullstack Developer

This was a personal project to learn more about the design part of React. I always learned React, but never took the change to learn more about designing a web page.

It is like Spotify. You can create an account. Log in and listen to music. Pretty easy, but in the back there is a lot going on.

Lab results would be generated by the system in PDF form and automatically sent to the client along with an interface for customers to be able to login and view.

All the users data and wesbite data are stored on Google Firebase. I use Firebase a lot. It is really nice to work with and I'd really like it.

I worked about 2 weeks on this project. I now don't use it anymore and it is not for public use.

Original product commercials version 1.0 and 2.0

Technologies:

  • - (S)CSS
  • - React JS
  • - SocketIO + Express
  • - React Smooth Scroll
  • - Google Firebase
